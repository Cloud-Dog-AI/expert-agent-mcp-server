# Query Responses Folder

## Purpose

This folder is for application delivery/runtime query responses.

**⚠️ Note:** This folder is version controlled but contains transient/temporary files.

## What Goes Here

- Cached query responses
- Runtime query result files
- Temporary response data

## Maintenance

- Response files are transient and can be regenerated
- May be cleared periodically
