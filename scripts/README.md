# Scripts Folder

## Purpose

This folder contains application solution scripts that are not related to project build/test, but are required as part of the delivery. Typically shell scripts.

## What Goes Here

- Deployment scripts
- Utility scripts
- Maintenance scripts
- Application-specific scripts (not build/test related)

## Example Scripts

- `docker-build.sh` - Docker image build script
- Server control scripts (if not in root)
- Deployment automation scripts
